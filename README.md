🚀 Featured Projects
1️⃣ Exploratory Data Analysis — Titanic Dataset
📌 Project Summary

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to uncover patterns behind passenger survival.

The analysis focuses on data cleaning, statistical exploration, feature engineering, and visualization to understand which factors influenced survival.

🔎 Key Questions

Did gender influence survival rates?

Did passenger class affect survival chances?

How did age, fare, and family size impact survival?

📊 Key Insights

Women had significantly higher survival rates than men

First-class passengers had better survival chances

Higher ticket fares correlated with survival
